# 校园二手教材交易平台 - 前端项目

## 快速开始

### 1. 安装依赖
cd frontend
npm install

### 2. 启动开发服务器
npm run dev

### 3. 访问地址
- 本地: http://localhost:5173
- 服务器: http://146.56.198.222:5173

## 项目说明

- 端口: 5173 (避免与3000端口冲突)
- 已完成首页 HomePage.vue 的完整样式
- 其他页面为占位页面,后续开发

## 开发命令

- `npm run dev` - 启动开发服务器
- `npm run build` - 构建生产版本
- `npm run preview` - 预览生产版本

## 技术栈

- Vue 3
- Vue Router
- Pinia
- Tailwind CSS
- Vite

## 注意事项

1. 开发服务器会自动打开浏览器
2. 服务器部署时需要配置Nginx反向代理
3. 确保服务器防火墙开放5173端口
